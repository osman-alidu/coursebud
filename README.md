# coursebud
To run module, run the ff cmds in command line:
1. For mac/Linux:
  - cd src 
  - python -m venv venv
  - source ./venv/bin/activate
  - pip3 install -r requirements.txt
  - python3 app.py

2. For Windows:
  - cd src 
  - python -m venv venv
  - .\venv\Scripts\Activate.ps1
  - pip3 install -r requirements.txt
  - python3 app.py
